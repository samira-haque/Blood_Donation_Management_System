Blood Donation Management System

A command-line based Blood Donation Management System developed using Bash Shell Scripting. This project provides an efficient way to manage blood donor information, blood bank records, emergency requests, and search operations through a modular and menu-driven interface.

Features
Add and manage donor records
Blood bank management
Emergency blood request handling
Authentication system
Search donor information
Dashboard and menu navigation
Modular shell scripting architecture
